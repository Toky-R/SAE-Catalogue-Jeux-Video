# SAE-Catalogue-Jeux-Video

SAE INFOA1 / INFO1 - Sup Galilee
Application web dynamique de gestion d'un catalogue de jeux video.

I) Presentation

Notre catalogue de jeux video permet de consulter, filtrer, ajouter et supprimer des jeux video depuis une interface web moderne. L'application repose sur une base de donnees MySQL interrogee en PHP via PDO.

Les donnees initiales proviennent de sources ouvertes et fiables.
